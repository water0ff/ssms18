# ssms18

Chocolatey package for **SQL Server Management Studio 18 (SSMS)**.

## Install

choco install ssms18 --version=18.0.0 -y
Notes
Uses the official Microsoft installer.

Installation is unattended.

Package maintained by the community (not Microsoft).
